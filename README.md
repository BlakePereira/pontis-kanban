## Deployment

This app is designed to deploy on Vercel, Railway, or similar platforms.

### Environment Variables Required

```
ADMIN_PASSWORD=your-secure-password
API_KEY=your-api-key-for-automation
SESSION_SECRET=your-session-secret
NODE_ENV=production
```

### Quick Deploy to Vercel

1. Connect this repo to Vercel
2. Set the environment variables above
3. Deploy!

## Usage

- **Web Access**: Visit the deployed URL, enter password to access the kanban board
- **API Access**: Use the API key in `X-API-Key` header for automated task management
